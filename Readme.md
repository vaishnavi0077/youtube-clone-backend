# 📺 YouTube Clone Backend

A scalable backend application inspired by YouTube, built using Node.js, Express.js, and MongoDB. The project provides secure authentication, video management, subscriptions, playlists, comments, likes, tweets, and dashboard analytics through a RESTful API architecture.

👩‍💻 **Developed By:** Vaishnavi Garad

🎓 Bachelor of Engineering (Computer Engineering)

🔗 **GitHub Repository:** https://github.com/vaishnavi0077/youtube-clone-backend

---

## 📌 Project Overview

YouTube Clone Backend is a backend-focused application developed to replicate the core functionalities of a modern video-sharing platform.

The project includes user authentication, video management, subscriptions, playlists, comments, likes, tweets, and dashboard analytics while following industry-standard backend development practices.

The primary goal of this project was to strengthen my understanding of scalable backend architecture, database relationships, REST API design, authentication systems, and MongoDB aggregation pipelines.
